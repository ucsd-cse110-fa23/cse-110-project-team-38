Q1.1: What is the purpose of the TaskList class?

    A: Container to hold all of the Task nodes

Q1.2: What is the purpose of the AppFrame class?

    A: container to hold all UI containers, and their defined positions (top, bottom, etc.)

Q1.3: What is the purpose of the addListeners method in the AppFrame class?

    A: Make it so that when a specific action on a button is performed, the intended function is called

Q1.4: What is the purpose of the start method in the Main class?

    A: to start the application. It calls all initialization functions via new AppFrame call.
